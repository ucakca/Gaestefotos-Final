# Zugriffsprobleme lösen

## Event nicht gefunden

### QR-Code funktioniert nicht
1. Prüfe, ob der Code beschädigt oder unscharf ist
2. Nutze die normale Kamera-App zum Scannen
3. Bitte den Host um den direkten Link

### Link funktioniert nicht
- Prüfe, ob der Link vollständig kopiert wurde
- Stelle sicher, dass du online bist
- Frage den Host, ob das Event noch aktiv ist

## "Event ist nicht mehr aktiv"

Das Event wurde vom Host beendet:
- Keine neuen Uploads möglich
- Fotos können noch angesehen werden
- Kontaktiere den Host für Zugang

## Seite lädt nicht

### 1. Verbindung prüfen
- WLAN oder mobile Daten aktiv?
- Andere Webseiten funktionieren?

### 2. Cache leeren
- **iPhone Safari:** Einstellungen → Safari → Verlauf löschen
- **Android Chrome:** Drei Punkte → Verlauf → Browserdaten löschen

### 3. Browser wechseln
- Versuche Chrome, Safari oder Firefox
- Meide alte Browser-Versionen

## Passwort-geschütztes Event

Manche Events erfordern ein Passwort:
1. Gib das Passwort ein, das dir der Host gegeben hat
2. Das Passwort ist oft auf der Einladung oder am Event-Ort
3. Frage den Host, wenn du es nicht weißt

## Als Host: Ich kann mein Event nicht bearbeiten

1. Stelle sicher, dass du eingeloggt bist
2. Prüfe, ob du der Event-Host bist
3. Bei Co-Host: Prüfe deine Berechtigungen
4. Versuche, dich neu einzuloggen
