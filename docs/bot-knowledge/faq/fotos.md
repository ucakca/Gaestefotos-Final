# Fotos - Häufige Fragen

## Wie lösche ich ein Foto?

Als Host kannst du jedes Foto in deinem Event löschen:
1. Öffne die Galerie deines Events
2. Tippe auf das Foto, das du löschen möchtest
3. Klicke auf das Papierkorb-Symbol oben rechts
4. Bestätige die Löschung

**Hinweis:** Gelöschte Fotos können nicht wiederhergestellt werden.

## Wie lade ich Fotos herunter?

**Einzelnes Foto:**
- Öffne das Foto in der Vollansicht
- Klicke auf das Download-Symbol

**Alle Fotos:**
- Gehe zu deinem Event-Dashboard
- Klicke auf "Alle Fotos herunterladen"
- Die Fotos werden als ZIP-Datei heruntergeladen

## Warum sehe ich manche Fotos nicht?

Mögliche Gründe:
- Das Foto wird noch hochgeladen (Balken sichtbar)
- Das Foto wurde von einem anderen Host gelöscht
- Internetverbindung unterbrochen während des Uploads

## Wie viele Fotos können hochgeladen werden?

- **Standard:** Unbegrenzt viele Fotos pro Event
- **Dateigröße:** Max. 50 MB pro Foto
- **Formate:** JPG, PNG, HEIC, WebP

## Wie kann ich Fotos in Alben sortieren?

1. Öffne die Galerie
2. Wähle die Fotos aus (lange drücken)
3. Tippe auf "Zu Album hinzufügen"
4. Wähle ein bestehendes Album oder erstelle ein neues
