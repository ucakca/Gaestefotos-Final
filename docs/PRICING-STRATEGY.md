## 5. Strategische Entscheidungen

| Entscheidung | Begründung |
|---|---|
| **Face Search für alle (inkl. Free)** | Alleinstellungsmerkmal, kein Konkurrent bietet das kostenlos |
| **Smart-Paket gratis bei jedem Add-on** | Massiver Mehrwert vs. Konkurrenz, macht Plattform "sticky" |
| **Keine Speicherbegrenzung** | Konkurrenz bietet unbegrenzt — muss mindestens gleichziehen |
| **Keine ZIP-Mengenbegrenzung** | Unnötige Komplexität, kein Kundenwert |
| **Partner-Abo pro Gerät** | Fair, skalierbar, einfach verständlich |
| **180 Tage max. Galerie** | Längere Laufzeiten machen keinen Sinn, spart Speicherkosten |
| **Fremdwerbung nur bei Free/Basic/Smart** | Monetarisierung der kostenlosen Nutzer, Anreiz für Premium |
