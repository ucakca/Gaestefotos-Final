# Teil 1 — Gäste

## 1. Als Gast: Das erste Mal

### QR-Code scannen
Der Veranstalter hat am Event QR-Codes aufgestellt oder per Einladung verschickt. So geht's:

1. **QR-Code scannen** mit der Kamera-App deines Smartphones  
2. Du wirst automatisch auf die **Event-Galerie** weitergeleitet — kein Download, kein Login nötig
3. Falls das Event **passwortgeschützt** ist, gibst du das Passwort ein (vom Veranstalter erhalten)

### Name eingeben
Beim ersten Upload wirst du nach deinem Namen gefragt:
- Dein Name erscheint bei deinen Fotos in der Galerie
- Er wird auf deinem Gerät gespeichert (kein Account nötig)
- Du kannst ihn jederzeit ändern

### Sprachauswahl
- Oben rechts findest du ein Sprach-Symbol 🌐
- Verfügbare Sprachen: Deutsch, Englisch, Französisch, Spanisch, Italienisch

## 2. Fotos & Videos hochladen

### Upload starten
1. Tippe auf den **Upload-Button** (Kamera-Symbol, unten rechts oder oben)
2. Wähle **Fotos aus deiner Galerie** oder mache ein Foto direkt
3. Der Upload startet automatisch — du siehst einen **Fortschrittsbalken**

### Mehrere Fotos auf einmal
- Halte beim Auswählen mehrere Fotos gleichzeitig ausgewählt
- Alle werden nacheinander hochgeladen

### Upload unterbrochen?
Kein Problem! Der Upload ist **resumable**:
- Verliere die WLAN-Verbindung? → Beim nächsten Versuch geht es genau dort weiter, wo es aufgehört hat
- Das Gerät sperren? → Upload läuft im Hintergrund weiter

### Upload-Limit
Manche Events haben ein **Limit für Uploads pro Gast** (z.B. "Maximal 20 Fotos").  
Du siehst dein aktuelles Kontingent im Upload-Dialog:
> *"7 / 20 Fotos hochgeladen — noch 13 möglich"*

### Was passiert nach dem Upload?
1. Dein Foto erscheint **sofort als kleine Vorschau** in der Galerie (Quick-Preview)
2. Kurz danach: **Vollauflösung** verfügbar (einige Sekunden)
3. Falls **Moderation aktiv:** Das Foto erscheint erst nach Freigabe durch den Veranstalter

### Häufige Upload-Fragen

**Mein Foto wurde abgelehnt — warum?**
- **Unschärfe:** Das Foto ist zu unscharf (automatische Erkennung) → Mach ein neues, schärferes Foto
- **Duplikat:** Das Foto wurde bereits hochgeladen (exakt oder sehr ähnlich)
- **Limit erreicht:** Das Event-Limit ist voll
- **Moderation:** Der Veranstalter hat es noch nicht freigegeben

**Welche Formate werden unterstützt?**
- Fotos: JPEG, PNG, HEIC, WebP
- Videos: MP4, MOV, AVI (max. 100 MB)

## 3. Die Galerie entdecken

### Galerie-Ansichten
- **Raster:** Alle Fotos in einem übersichtlichen Grid
- **Timeline:** Nach Uhrzeit sortiert
- **Kategorien/Alben:** Vom Veranstalter angelegte Sammlungen
- **Meine Fotos:** Nur deine eigenen Uploads

### Fotos ansehen
- Tippe auf ein Foto um es in **Vollansicht** zu öffnen
- **Wischen:** zum nächsten/vorherigen Foto
- **Pinch-to-Zoom:** Bild heranzoomen

### Reagieren
- **Herz ❤️:** Foto liken (falls aktiviert)
- **Kommentar 💬:** Kommentar hinterlassen (falls aktiviert)
- **Teilen 📤:** Foto teilen (Social Media, WhatsApp, etc.)

### Gästebuch
Falls vom Veranstalter aktiviert:
- Tippe auf **"Gästebuch"** im Menü
- Schreibe eine Nachricht mit Text, Emoji oder Foto
- Dein Eintrag erscheint in der digitalen Chronik des Events

### Gesuchssuche (Face Search)
Falls aktiviert:
1. Mache ein Selfie
2. Die App findet **alle Fotos auf denen du drauf bist** — auch von anderen Gästen
3. Diese Funktion nutzt biometrische Analyse (Einwilligung erforderlich)

### Live Wall
Falls aktiviert: Fotos erscheinen in Echtzeit auf einer großen Leinwand am Event.

## 4. KI-Effekte & Spiele

### KI-Bild-Effekte
So wendest du einen KI-Effekt auf ein Foto an:

1. Öffne ein Foto in der Vollansicht
2. Tippe auf **"KI-Effekte"** (Zauberstab-Symbol ✨)
3. Wähle einen Effekt (z.B. Cartoon, Ölgemälde, Anime, Vintage...)
4. Warte kurz (ca. 25–35 Sekunden)
5. Das KI-Bild erscheint — du kannst es **speichern oder teilen**

**Verfügbare Stil-Effekte (Auswahl):**
| Effekt | Beschreibung |
|--------|-------------|
| Cartoon | Pixar-ähnlicher 3D-Cartoon-Stil |

| Renaissance | Altes Meister-Portrait |

2. Wähle das Ziel-Foto (dein Gesicht) und das Vorlage-Foto
3. Die KI tauscht das Gesicht aus (ca. 25–35 Sekunden)

Tippe auf **"Spiele"** oder **"AI-Games"** im Menü:

| und mehr... | |

### Credits-System
Manche Events nutzen ein Credit-System für KI-Features:
- Jeder KI-Effekt kostet eine bestimmte Anzahl Credits
- Der Veranstalter legt das Budget fest
- Du siehst deine verfügbaren Credits im Menü

## 5. Fotos teilen & herunterladen

### Einzelnes Foto herunterladen
1. Foto in Vollansicht öffnen
2. Tippe auf das **Download-Symbol ⬇️**
3. Das Foto wird in deiner Galerie gespeichert

### Per E-Mail teilen
1. Foto öffnen → Teilen-Symbol → **"Per E-Mail"**
2. E-Mail-Adresse eingeben
3. Du erhältst einen Download-Link

### Alle eigenen Fotos
- Gehe zu **"Meine Fotos"**
- Tippe auf **"Alle herunterladen"** (falls verfügbar)

### QR-Code für direkten Link
- Jedes Foto hat einen QR-Code zum direkten Teilen
- Tippe auf das Teilen-Symbol → **"QR-Code"**

# Teil 2 — Hosts (Veranstalter)

## 6. Account erstellen & Paket wählen

### Account registrieren
1. Gehe zu [gästefotos.com](https://gästefotos.com)
2. Klicke auf **"Jetzt kostenlos starten"**
3. E-Mail + Passwort eingeben
4. E-Mail bestätigen

### Paket wählen
| Paket | Events | Fotos/Event | KI-Features | Storage |
|-------|--------|-------------|-------------|---------|
| **Free** | 1 | 50 | Basis | 14 Tage |
| **Basic** | 3 | 500 | Standard | 30 Tage |
| **Smart** | 10 | 2.000 | Pro | 90 Tage |
| **Premium** | unbegrenzt | unbegrenzt | Alle | 365 Tage |

**Paket kaufen:** gästefotos.com → Shop → Paket auswählen → Zahlung via WooCommerce

### Dashboard aufrufen
1. Melde dich an auf [app.gästefotos.com](https://app.gästefotos.com)
2. Du wirst automatisch zum **Host-Dashboard** weitergeleitet
3. Alternativ: Oben rechts → dein Name → "Dashboard"

## 7. Event erstellen & konfigurieren

### Neues Event anlegen
1. Host-Dashboard → **"Neues Event"**
2. Pflichtfelder ausfüllen:
   - **Eventname** (z.B. "Hochzeit Müller-Schmidt")
   - **Datum** (Anfang und Ende)
   - **Event-Typ** (Hochzeit, Geburtstag, Firmenevent, etc.)
3. **"Event erstellen"** klicken

### Event konfigurieren

**Grundeinstellungen** (Event → Einstellungen):

| Einstellung | Beschreibung | Standard |
|-------------|-------------|---------|
| Passwortschutz | Gäste müssen Passwort eingeben | Aus |
| Moderation | Fotos erst nach Freigabe sichtbar | Aus |
| Kommentare | Gäste können kommentieren | Ein |
| Likes | Gäste können liken | Ein |
| Max. Uploads/Gast | Limit pro Gast (leer = unbegrenzt) | — |
| Gästebuch | Digitales Gästebuch aktivieren | Aus |
| Face Search | Gesichtserkennung aktivieren | Aus |
| Live Wall | Echtzeit-Galerie-Anzeige | Aus |

**Design & Branding**:
- Cover-Foto hochladen
- Farbschema anpassen
- Event-Logo hochladen

**KI-Features konfigurieren** (Event → KI-Einstellungen):
- Welche KI-Effekte sind für Gäste verfügbar?
- Credits pro Gast festlegen
- Bestimmte Effekte deaktivieren

### Event-URL
Nach dem Erstellen erhältst du eine URL:
```
https://app.gästefotos.com/e/{dein-event-slug}
```
Diese URL teilst du mit deinen Gästen (oder als QR-Code).

## 8. QR-Code & Einladungen

### QR-Code generieren
1. Event → **"QR-Code"** Tab
2. QR-Code anpassen (Farbe, Logo, Rahmen-Design)
3. **Exportieren** als:
   - PNG (für digitale Nutzung)
   - PDF (für Druck — A4, A5, Postkarte)
   - Aufsteller-Template (fertig zum Drucken)

### QR-Code-Templates
Vorgefertigte Designs für verschiedene Anlässe:
- Hochzeits-Aufsteller (elegant, weiß/gold)
- Party-Aufsteller (bunt, modern)
- Firmen-Template (sachlich, mit Logo-Platz)
- Tischkärtchen (klein, 4-er Format)

### Einladungen versenden
Event → **"Einladungen"**:
- **E-Mail-Einladung:** Vorlage auswählen, E-Mails eingeben, versenden
- **WhatsApp-Text:** Fertiger Text mit Link zum Kopieren
- **Print-Einladung:** PDF-Vorlage mit QR-Code

## 9. Galerie moderieren

### Moderation aktivieren
Event → Einstellungen → **"Freigabe erforderlich"** aktivieren  
→ Neue Fotos sind zunächst nicht öffentlich sichtbar

### Fotos freigeben oder ablehnen
Host-Dashboard → **"Zu moderieren"** Tab:
- ✅ **Freigeben:** Foto wird sofort in der Galerie sichtbar
- ❌ **Ablehnen:** Foto wird nicht angezeigt (bleibt aber in DB)
- **Alle freigeben:** Alle wartenden Fotos mit einem Klick freigeben

### Fotos melden
Gäste können Fotos melden (unpassende Inhalte):
- Host-Dashboard → **"Gemeldete Fotos"**
- Entscheiden: Freigeben oder löschen

### Fotos löschen
1. Foto in der Host-Galerie anklicken
2. Drei-Punkte-Menü → **"Löschen"**
3. Bestätigen

### Alben & Kategorien
- Host-Dashboard → **"Alben"** → **"Album erstellen"**
- Fotos per Drag & Drop in Alben sortieren
- KI kann Alben automatisch vorschlagen (Smart Album)

## 12. Co-Hosts einladen

Co-Hosts können bei einem Event helfen (moderieren, verwalten):

1. Host-Dashboard → Event → **"Team"** Tab
2. **"Co-Host einladen"** → E-Mail eingeben
3. Co-Host erhält Einladungs-E-Mail
4. Nach Annahme: Co-Host kann Event mitverwalten

**Co-Host kann:**
- Fotos moderieren und löschen
- Alben verwalten
- Galerie einsehen
- QR-Code herunterladen

**Co-Host kann NICHT:**
- Event löschen
- Paket ändern
- Co-Hosts hinzufügen/entfernen

## 15. KI-Konfiguration

### AI-Provider verwalten
Admin → **"KI-System"** → **"Provider"** (`/manage/ai-providers`):

| Provider | Zweck | Konfiguration |
|----------|-------|---------------|
| RunPod | Bild-Effekte (primär) | API-Key, Endpoint-ID |
| fal.ai | Bild-Fallback, Video | API-Key |
| Groq | LLM (primär) | API-Key |
| Grok/xAI | LLM-Fallback | API-Key |
| OpenAI | LLM-Fallback 2 | API-Key |
| remove.bg | Background Removal | API-Key |
| Ollama | LLM offline | URL (lokal) |

**Status prüfen:** Grüner Punkt = Provider erreichbar, Roter Punkt = offline/Fehler

### ComfyUI Workflows verwalten
Admin → **"KI-System"** → **"Workflows"** (`/manage/ai-workflows`):

Für jeden der 18 Workflows kannst du:
- **JSON anzeigen:** Aktuellen Workflow-Code einsehen
- **JSON aktualisieren:** Neuen ComfyUI-Workflow hochladen
- **Test-Job:** Effekt mit einem Beispielbild testen
- **Zurücksetzen:** Auf Default-Workflow zurücksetzen

> **Hinweis:** Workflows sind ComfyUI-API-Format-JSONs. Exportiere sie aus ComfyUI (`Save (API Format)`).

### Prompt-Templates verwalten
Admin → **"KI-System"** → **"Prompt-Templates"** (`/manage/prompt-templates`):
- LLM-Prompts für Spiele, Vorschläge, Chat bearbeiten
- Live-Preview: Prompt sofort testen
- Versionierung: Änderungen werden gespeichert

### AI-Features konfigurieren
Admin → **"KI-System"** → **"AI-Features"** (`/manage/ai-features`):
- Feature ↔ Provider Zuordnung ändern
- Features system-weit deaktivieren
- Credit-Kosten pro Feature anpassen

### AI-Logs einsehen
Admin → **"KI-System"** → **"Logs"** (`/manage/ai-logs`):
- Alle AI-Jobs mit Status (DONE/FAILED)
- Fehler-Details einsehen
- Jobs manuell neu starten

- Partner-spezifische Provider nutzen die Partner-Keys (nicht die System-Keys)

### Impersonation (Als Benutzer einloggen)
Für Support-Zwecke:
1. Admin → Benutzer → Benutzer anklicken
2. **"Als dieser Benutzer einloggen"**
3. Du siehst die App genau wie dieser Benutzer
4. Oben erscheint ein oranger Hinweis "Impersonation aktiv"
5. **"Impersonation beenden"** um zurückzukehren

> **Sicherheitshinweis:** Impersonation wird im Audit-Log protokolliert.

### Host-Checkliste für den Event-Tag
- [ ] Event-Passwort notiert (falls aktiviert)?
- [ ] QR-Code-Aufsteller ausgedruckt und aufgestellt?
- [ ] Test-Upload gemacht (Galerie funktioniert)?
- [ ] Moderation-Einstellung geprüft (auto oder manuell)?
- [ ] KI-Effekte aktiviert (falls gewünscht)?
- [ ] Co-Hosts informiert und eingeloggt?

### Admin-Checkliste für Systemstart
- [ ] `curl http://localhost:8001/api/health` → HTTP 200?
- [ ] Redis aktiv? `redis-cli ping` → PONG
- [ ] PostgreSQL aktiv? `systemctl status postgresql`
- [ ] SeaweedFS aktiv? `curl http://localhost:8333/dir/status`
- [ ] AI-Provider grün in `/manage/ai-providers`?
- [ ] Kein Wartungsmodus aktiv?
