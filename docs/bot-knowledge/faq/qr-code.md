# QR-Code - Häufige Fragen

## Wie funktioniert der QR-Code?

Der QR-Code ist der einfachste Weg für Gäste:
1. Gast scannt den Code mit der Handy-Kamera
2. Die Event-Seite öffnet sich automatisch
3. Gast kann sofort Fotos hochladen

**Kein App-Download nötig!**

## Wo finde ich meinen QR-Code?

1. Öffne dein Event-Dashboard
2. Klicke auf "QR-Code" im Menü
3. Der Code wird angezeigt

## Wie kann ich den QR-Code herunterladen?

Im QR-Code-Bereich findest du Download-Optionen:
- **PNG** - Für digitale Nutzung
- **PDF** - Zum Drucken (A4)
- **SVG** - Für professionellen Druck

## Kann ich den QR-Code gestalten?

Ja! Im QR-Styler kannst du:
- Farben anpassen
- Logo hinzufügen
- Rahmen und Text ändern
- Verschiedene Stile wählen

## Wie drucke ich den QR-Code am besten?

**Empfehlungen:**
- Mindestgröße: 3x3 cm
- Hoher Kontrast (dunkel auf hell)
- Auf mattem Papier (weniger Reflexionen)

**Ideen für die Platzierung:**
- Tischkarten
- Einladungen
- Poster am Eingang
- Menükarten

## Der QR-Code funktioniert nicht - was tun?

1. Prüfe, ob der Code groß genug ist
2. Stelle sicher, dass er nicht verzerrt ist
3. Teste mit der normalen Kamera-App
4. Prüfe die Internetverbindung
